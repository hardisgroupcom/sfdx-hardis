## Get org info, instance info, and next major upgrade date

This check retrieves key org-level and Salesforce instance-level information, including the next major release window.

It helps teams anticipate platform changes and prepare releases accordingly.

Sfdx-hardis command: [sf hardis:org:diagnose:instanceupgrade](https://sfdx-hardis.cloudity.com/hardis/org/diagnose/instanceupgrade/)

Key: **ORG_INFO**
