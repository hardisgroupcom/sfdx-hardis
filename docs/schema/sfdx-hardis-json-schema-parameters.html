sfdx-hardis configuration

Type: object

Configuration file definition for sfdx-hardis Salesforce DX plugin

 No Additional Properties

root promptsLanguage

Prompts language(s)

Type: string Default: "en"

Language used for AI-generated prompts. If not set, the default language is English (en). Supported languages are defined in the code but not limited to: English (en), French (fr), Spanish (es), German (de), Italian (it), Portuguese (pt). If multiple languages, separate them with comma (ex: "en,fr,de") and the first one will be used as default language for prompts. (mirrors PROMPTS_LANGUAGE environment variable)

root promptsParallelCallNumber

Prompts parallel call number

Type: number Default: 10

Number of parallel AI prompt calls when generating project documentation. Higher values speed up documentation generation but may hit API rate limits. (mirrors PROMPTSPARALLELCALL_NUMBER environment variable)

Value must be greater or equal to 1


Examples:

1

5

10

root useLangchainLlm

Enable LangChain

Type: boolean Default: false

Set to true to enable LangChain integration (mirrors the USELANGCHAINLLM environment variable).

root langchainLlmProvider

LangChain provider

Type: enum (of string)

LLM provider identifier used with LangChain (ex: openai, anthropic, google-genai, ollama). Equivalent to LANGCHAINLLMPROVIDER env var.

Must be one of:

  • "openai"
  • "anthropic"
  • "google-genai"
  • "ollama"

root langchainLlmModel

LangChain model

Type: string

Model identifier to use with the configured LangChain provider. Latest models: OpenAI (gpt-5.2, gpt-5.2-mini, gpt-5, gpt-4.1, gpt-4o, gpt-4o-mini...), Anthropic (claude-opus-4-6, claude-3-5-sonnet, claude-3-5-haiku, claude-3-haiku...), Google Gemini (gemini-3-pro, gemini-3-pro-deep-think, gemini-3-flash, gemini-2.5-flash-lite...). Equivalent to LANGCHAINLLMMODEL env var.

root langchainLlmTemperature

LangChain temperature

Type: number

Sampling temperature override for LangChain requests (maps to LANGCHAINLLMTEMPERATURE env var).

root langchainLlmMaxTokens

LangChain max tokens

Type: number

Maximum number of tokens generated per LangChain response (maps to LANGCHAINLLMMAX_TOKENS env var).

root langchainLlmMaxRetries

LangChain max retries

Type: number

Maximum number of retries for failed LangChain requests (maps to LANGCHAINLLMMAX_RETRIES env var).

root langchainLlmTimeout

LangChain timeout

Type: number

Timeout in milliseconds applied to LangChain calls (maps to LANGCHAINLLMTIMEOUT env var).

root langchainLlmBaseUrl

LangChain base URL

Type: string

Custom base URL for LangChain HTTP-based providers (for example a local Ollama endpoint). Equivalent to LANGCHAINLLMBASE_URL env var.

Type: string

[
    {
        "id": "knowledgeUnassign",
        "label": "Remove KnowledgeUser right to the user who has it",
        "type": "command",
        "command": "sf data update record --sobject User --where \"UserPermissionsKnowledgeUser='true'\" --values \"UserPermissionsKnowledgeUser='false'\" --json",
        "context": "all"
    },
    {
        "id": "importDataPackage",
        "label": "Import SFDMU data package",
        "type": "data",
        "parameters": {
            "sfdmuProject": "EmailTemplate"
        },
        "context": "process-deployment-only"
    }
