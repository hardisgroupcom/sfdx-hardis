# Network issue

## Detection

- String: `ECONNABORTED`
- String: `ECONNRESET`

## Resolution

```shell
The network connection has been aborted, this is a purely technical issue.
Try again, and if you still see errors, check the status of Salesforce instance on https://status.salesforce.com
```
