## Detect Apex and Flow errors

This check helps detect Apex exceptions and Flow execution errors so issues can be triaged quickly.

Use this report to identify recurring failures and prioritize fixes before they impact business users.

Sfdx-hardis command: [sf hardis:org:monitor:errors](https://sfdx-hardis.cloudity.com/hardis/org/monitor/errors/)

Key: **APEX_FLOW_ERRORS**
