## Community Support - GitHub Issues

Basic support is provided free of charge through the **GitHub Issues tracker** of the sfdx-hardis repository.

If you encounter a bug, have a question, or want to suggest an improvement:

1. Search [existing issues](https://github.com/hardisgroupcom/sfdx-hardis/issues){target=blank} to see if your topic is already covered.
2. Open a [new issue](https://github.com/hardisgroupcom/sfdx-hardis/issues/new/choose){target=blank} with as much detail as possible (sfdx-hardis version, OS, command run, output).

Community support is best-effort and handled by maintainers and contributors in their available time.
