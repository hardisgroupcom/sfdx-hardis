# Deployment Agent

Deployment errors are common and quite boring, right?

sfdx-hardis Deployment Agent helps you solve them, even if you are not using sfdx-hardis CI/CD pipelines.

It contains core rules and can optionally be integrated with AI to provide the best guidance.

![](assets/images/AI-Assistant.gif)

## Integrations

Deployment Agent provides tips in Pull Request comments (GitHub, Gitlab, Azure, Bitbucket).

It also provides tips in console logs.

![](assets/images/AI-deployment-assistant-console.png)
